19.1. How to get the list of interfaces for a specific device?#

Caught in Mantis #455627.

REQUEST:

{
  "id": 1,
  "method": "get",
  "params": [
    {
      "url": "/dbcache/system/interface",
      "option": [
        "scope member"
      ],
      "scope member": [
        {
          "name": "FGT60Da",
          "vdom": "global"
        }
      ],
      "fields": [
        "name",
        "alias",
        "ip",
        "ipv6",
        "mode",
        "type",
        "vdom",
        "status"
      ],
      "current_adom": "455627_mapping"
    }
  ]
}
